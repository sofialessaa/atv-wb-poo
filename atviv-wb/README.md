# Rodando a atviv-wb
Execute comandos para rodar o frontend e backend.
<hr>

## Rodando o Frontend
##### 1. Abra o terminal powershell
##### 2. `cd atviv-wb`
##### 3. `cd frontend`
##### 4. `npm install`
##### 5. `npm start`
